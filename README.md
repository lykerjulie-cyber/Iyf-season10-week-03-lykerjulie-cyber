## What I Learned

1. **Command Line**: Navigating file systems, creating/moving files, and searching with `find` and `grep` without using GUI.
2. **Git Branching**: Creating feature branches, resolving merge conflicts, and maintaining clean commit history.
3. **Shell Scripting**: Automating project setup with bash scripts to save time on repetitive tasks.
4. **GitHub Workflow**: Using `.gitignore`, pushing/pulling, and writing conventional commit messages.
